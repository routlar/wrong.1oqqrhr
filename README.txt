RIATTAGOTCHI BUILD INSTRUCTIONS
1. Open the 'assets' folder.
2. Place your images (Cat.jpeg, CatStart.jpeg, etc.) and sounds (Song.mp3, meow.mp3, etc.) inside 'assets/'.
3. Place your font file (Font.ttf) in the main folder alongside index.html.
4. To run, use a local server (like Live Server in VS Code) to avoid CORS browser errors.
